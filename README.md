# 💸 Expense Tracker

A clean and intuitive expense tracking app built with Flutter. Log your expenses, visualize your spending, and stay on top of your budget!

## ✨ Features

- 💰 Add and delete expenses with ease
- 📊 Visual bar chart to see spending by category
- 🗂️ Organized expense list with categories
- 🔄 Swipe to delete expenses
- 📱 Clean, minimal UI built with Flutter
- 📐 Fully responsive layout for both portrait and landscape orientations

---

## 🏗️ Project Structure

```
lib/
├── main.dart                        # App entry point
├── models/
│   └── expense.dart                 # Expense data model
└── widgets/
    ├── chart/
    │   ├── chart.dart               # Spending chart widget
    │   └── chart_bar.dart           # Individual chart bar widget
    ├── expenses_list/
    │   ├── expenses_list.dart       # Scrollable list of expenses
    │   └── expenses_item.dart       # Single expense item widget
    ├── expenses.dart                # Main expenses screen
    └── new_expense.dart             # Add new expense form
```
